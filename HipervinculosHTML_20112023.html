Hipervínculos o Links

Son elementos muy característicos de los diseños de páginas web, puesto que nos van a permitir que las páginas
se puedean relacionar entre sí de una manera sencilla y con libertad (independientemente de dónde se alojen)
Existen por lo tanto dos tipos de hipertexto o hipervínculos:
  • El enlace, es decir, el elemente texto o imagen sobre el que se puede hacer click con el ratón
  • El destino, es decir, el elemento (página web, imágen, archivo, etc.) que se muestra al hacer
    click en el enlace.

La estructura en la que se basa los hipervínculos es la siguiente.
  • Etiqueta de apertura que se llama anchor (anclaje). Viene definida por la etiqueta a de
    apertura y cierre.
  • Atributo. Este atributo es el href, o hace referencia al link al que se quiere ir. Siempre entre
    doble comillado de apertura y cierre. (".....").
  • El texto que va entre la etiqueta de apertura y cierre, que es el que utilizaremos como enlace
